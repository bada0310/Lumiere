# 맞춤 화장품 추천 리스트 페이지 (수진)

ERD 기준으로 `Product`, `ProductOption`, `PersonalColor`, `ColorMetric`, `Wishlist`가 다 연결되는 화면

중요한 점은 추천 대상이 제품 자체가 아니라 **제품 옵션**이라는 것. 예를 들어 “롬앤 틴트”가 아니라 “롬앤 틴트 23호 베어그레이프”가 추천 대상
