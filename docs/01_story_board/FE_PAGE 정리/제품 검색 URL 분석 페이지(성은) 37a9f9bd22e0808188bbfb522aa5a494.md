# 제품 검색 / URL 분석 페이지(성은)

이건 Lumière의 차별점이야. ERD의 `ExternalAnalysis`와 연결돼.

ExternalAnalysis는 사용자가 입력한 외부 제품 링크 분석 이력을 관리하는 엔터티야.

[화면 이미지](%ED%99%94%EB%A9%B4%20%EC%9D%B4%EB%AF%B8%EC%A7%80%2037a9f9bd22e080b5ab60c6ccab9a177e.md)

<들어갈 요소>

- 통합 검색창
브랜드명 / 제품명 검색
- URL 입력창
올리브영 URL 입력
- 분석하기 버튼
- 최근 분석한 제품
- 검색 결과 미리보기
- 분석 안내
“제품의 색상 옵션을 추출해 내 피부톤과 비교해요”

<연결>

검색어 입력 → 검색 결과
URL 입력 → URL 분석 결과
