# 로그인/회원가입 페이지 (성은 )

ERD의 `User`와 연결되는 화면이야. User는 이메일, 비밀번호, 닉네임, 프로필 이미지, 계정 상태 등을 가짐

[화면 이미지](%ED%99%94%EB%A9%B4%20%EC%9D%B4%EB%AF%B8%EC%A7%80%2037a9f9bd22e0809cafe7df6ba74c13c4.md)

<들어갈 요소>

- 이메일 입력
- 비밀번호 입력
- 로그인 버튼
- 회원가입 탭
- 소셜 로그인 버튼
- 비밀번호 찾기
- 개인정보 처리 안내
- 안면 이미지 저장하지 않는다는 안내

<연결>

로그인 성공 → 메인
회원가입 완료 → 메인
비회원 진단 시도 → 로그인 유도 또는 비회원 진단 허용 여부 선택
