## 1. 화면 연결성 정리

```jsx
[메인 페이지]
   ├─ 로그인 / 회원가입
   ├─ AI 진단 시작
   │     ↓
   │  사진 업로드
   │     ↓
   │  AI 분석 로딩
   │     ↓
   │  퍼스널컬러 진단 결과
   │     ├─ AI 메이크오버
   │     ├─ 맞춤 화장품 추천 리스트
   │     └─ 마이페이지 진단 기록 저장
   │
   ├─ 제품 검색 / URL 분석
   │     ↓
   │  URL 분석 결과
   │     ↓
   │  제품 상세 분석
   │     ├─ 찜하기
   │     ├─ 구매 아웃링크
   │     └─ 커뮤니티 후기
   │
   ├─ 추천 제품
   │     ↓
   │  제품 상세 분석
   │
   ├─ 톤 커뮤니티
   │     ↓
   │  제품 상세 분석
   │
   └─ 마이페이지
         ├─ 진단 기록 다시보기
         ├─ 찜한 제품 보기
         ├─ 최근 분석 기록 보기
         └─ 내가 쓴 글 보기
```

## 2. ERD와 화면 연결표

| ERD 엔터티 | 보여줘야 하는 화면 |
| --- | --- |
| User | 로그인, 회원가입, 마이페이지 |
| Diagnosis | 진단 결과, 마이페이지 진단 기록 |
| PersonalColor | 진단 결과, 추천 리스트 |
| Category | 추천 제품 카테고리 탭 |
| Product | 추천 리스트, 제품 상세 |
| ProductOption | 추천 리스트, URL 분석 결과, 제품 상세 |
| Wishlist | 추천 리스트 찜 버튼, 마이페이지 찜 목록 |
| ExternalAnalysis | URL 분석, URL 분석 결과, 마이페이지 최근 분석 |
| ColorMetric | 제품 상세 분석, 적합도 그래프, 추천 순위 |
