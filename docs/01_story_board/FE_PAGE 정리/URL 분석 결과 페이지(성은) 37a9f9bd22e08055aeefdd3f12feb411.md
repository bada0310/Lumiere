# URL 분석 결과 페이지(성은)

[화면 이미지](%ED%99%94%EB%A9%B4%20%EC%9D%B4%EB%AF%B8%EC%A7%80%2037a9f9bd22e0803b95cee4901dc95166.md)

<들어갈 요소>

- 입력한 URL 제품명
- 브랜드명
- 분석 상태
- 전체 색상/호수 리스트
- 내 피부톤과 가장 잘 맞는 호수 TOP 3
- BEST 태그
- 각 옵션별 적합도 %
- 색상칩
- 추천 이유
- 제품 상세 분석으로 이동 버튼

*`ExternalAnalysis` 속성에 `input_url`, `product_name`, `brand`, `analysis_status`가 있고, `ColorMetric`과 연결돼서 유사도 결과를 만들 수 있음. 

<연결>

URL 분석 결과 → 제품 상세 분석
URL 분석 결과 → 찜하기
URL 분석 결과 → 마이페이지 최근 분석 기록
