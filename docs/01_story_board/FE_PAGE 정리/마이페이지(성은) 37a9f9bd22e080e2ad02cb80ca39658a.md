# 마이페이지(성은)

ERD 기준으로 `User`, `Diagnosis`, `Wishlist`, `ExternalAnalysis`가 모두 보여야 하는 화면이야.

[화면 이미지](%ED%99%94%EB%A9%B4%20%EC%9D%B4%EB%AF%B8%EC%A7%80%2037a9f9bd22e0800bb931c7b860ef8fd9.md)

<들어갈 요소>

- 프로필 이미지
- 닉네임
- 이메일
- 정보 수정 버튼
- 나의 퍼스널컬러 요약
- 진단 기록 리스트
- 결과 다시보기 버튼
- 찜한 제품 옵션 목록
- 최근 URL 분석 기록
- 내가 쓴 커뮤니티 글
- 재진단하기 버튼

*User는 Diagnosis, Wishlist, ExternalAnalysis와 각각 1:N 관계야. 즉 한 사용자는 여러 진단 기록, 여러 찜 기록, 여러 URL 분석 기록을 가질 수 있음.

<연결>

마이페이지 → 과거 진단 결과
마이페이지 → 찜한 제품 상세
마이페이지 → 최근 URL 분석 결과
마이페이지 → 재진단
