# 진단 결과 삭제

Method: DELETE
URL: /api/diagnosis/results/{result_id}/
param: result_id
사용자: 회원
설명: 특정 진단 결과를 삭제합니다.
순서: 18
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
DELETE /api/diagnosis/results/{result_id}/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `result_id` | 확인 필요 | 확인 필요 | result_id 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |

### Request Body

요청 바디 없음.

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| deleted_id | 삭제된 진단 결과 ID | Number |  | X | 1 |
| was_primary | 삭제 전 메인 여부 | Boolean |  | X | true |
| new_primary_id | 새 메인 진단 결과 ID | Number |  | O | null |
| new_primary | 새 메인 진단 결과 | Object |  | O | null |
| message | 처리 결과 메시지 | String |  | X | "진단 결과가 삭제되었습니다." |

### Status

| status | response content |
| --- | --- |
| 200 | 삭제 성공 |
| 401 | 인증 실패 |
| 404 | 진단 결과 없음 |
| 500 | 서버 내부 오류 |

  "deleted_id": 1,
