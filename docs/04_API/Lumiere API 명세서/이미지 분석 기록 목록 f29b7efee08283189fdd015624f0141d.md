# 이미지 분석 기록 목록

Method: GET
URL: /api/products/image-analyses/
param: limit, page, page_size
사용자: 회원
설명: 내 상품 이미지 분석 기록 목록을 조회합니다.
순서: 30
인증필요여부: 로그인 필요
카테고리: 상품/추천 21

### Request Syntax

```jsx
GET /api/products/image-analyses/
```

### Parameters

| Name | Type | Required | Description |

| `page` | 확인 필요 | 확인 필요 | page 값 |
| `page_size` | 확인 필요 | 확인 필요 | page_size 값 |
