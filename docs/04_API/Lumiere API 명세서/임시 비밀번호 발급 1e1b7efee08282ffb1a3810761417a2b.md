# 임시 비밀번호 발급

Method: POST
URL: /accounts/find-password/
param: email
사용자: 비회원
설명: 가입 이메일로 임시 비밀번호를 발송합니다.
순서: 11
인증필요여부: 인증 불필요
카테고리: 계정 11

### Request Syntax

```jsx
POST /accounts/find-password/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `email` | 확인 필요 | 확인 필요 | email 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| email | 가입 이메일 | String |  | X | "user01@example.com" |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| message | 처리 결과 메시지 | String |  | X | "임시 비밀번호가 이메일로 발송되었습니다." |

### Status

| status | response content |
| --- | --- |
| 200 | 임시 비밀번호 발급 성공 |
| 400 | email 누락 또는 이메일 형식 오류 |
| 404 | 가입된 이메일 없음 |
| 500 | 서버 내부 오류 |

### Example

