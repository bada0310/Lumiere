# 이미지 분석 기록 수정

Method: PATCH
URL: /api/products/image-analyses/{analysis_id}/
param: analysis_id, brand_name, category, options, product_name
사용자: 회원
설명: 이미지 분석 결과의 상품명, 브랜드명, 카테고리, 색상 옵션을 수정합니다.
순서: 32
인증필요여부: 로그인 필요
카테고리: 상품/추천 21

### Request Syntax

```jsx
PATCH /api/products/image-analyses/{analysis_id}/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `analysis_id` | 확인 필요 | 확인 필요 | analysis_id 값 |
| `product_name` | 확인 필요 | 확인 필요 | product_name 값 |
| `brand_name` | 확인 필요 | 확인 필요 | brand_name 값 |
| `category` | 확인 필요 | 확인 필요 | category 값 |
| `options` | 확인 필요 | 확인 필요 | options 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| product_name | 상품명 | String | 선택 | O | "수정된 상품명" |
| brand_name | 브랜드명 | String | 선택 | O | "브랜드명" |
| category | 카테고리 | String(ENUM) | 선택 | O | "LIP" |
| options | 색상 옵션 목록 | Array | 선택 | O | [] |

### Response

수정된 이미지 분석 기록 객체.

### Status

| status | response content |
| --- | --- |
| 200 | 수정 성공 |
| 400 | 잘못된 입력 |
| 401 | 인증 실패 |
| 404 | 이미지 분석 기록 없음 |
| 500 | 서버 내부 오류 |
