# JWT 토큰 갱신

Method: POST
URL: /accounts/jwt-refresh/
param: refresh
사용자: 비회원
설명: 새 access 토큰을 발급합니다.
순서: 5
인증필요여부: 인증 불필요
카테고리: 계정 11

### Request Syntax

```jsx
POST /accounts/jwt-refresh/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `refresh` | 확인 필요 | 확인 필요 | refresh 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| refresh | refresh token | String |  | X | "refresh_token" |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| access | 새 access token | String |  | X | "new_access_token" |

### Status

| status | response content |
| --- | --- |
| 200 | 토큰 갱신 성공 |
| 400 | refresh token 누락 |
| 401 | refresh token 만료 또는 유효하지 않음 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "access": "new_access_token"
}
```
