# 닉네임 중복 확인

Method: POST
URL: /accounts/check-nickname/
param: nickname
사용자: 회원
설명: 닉네임 사용 가능 여부를 확인합니다.
순서: 10
인증필요여부: 로그인 필요
카테고리: 계정 11

### Request Syntax

```jsx
POST /accounts/check-nickname/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `nickname` | 확인 필요 | 확인 필요 | nickname 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| nickname | 확인할 닉네임 | String |  | X | "새닉네임" |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| is_available | 사용 가능 여부 | Boolean |  | X | true |
| message | 결과 메시지 | String |  | X | "사용 가능한 닉네임입니다." |

### Status

| status | response content |
| --- | --- |
| 200 | 확인 성공 |
| 400 | nickname 누락 |
| 401 | 인증 실패 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "is_available": true,
  "message": "사용 가능한 닉네임입니다."
}
```
