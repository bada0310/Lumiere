# 진단 결과 목록 조회

Method: GET
URL: /api/diagnosis/results/
param: limit, page, page_size
사용자: 회원
설명: 로그인 사용자의 진단 결과 목록을 조회합니다.
순서: 15
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
GET /api/diagnosis/results/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `limit` | 확인 필요 | 확인 필요 | limit 값 |
| `page` | 확인 필요 | 확인 필요 | page 값 |
| `page_size` | 확인 필요 | 확인 필요 | page_size 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |

| Authorization | O | JWT access token | Bearer {access_token} |

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| count | 전체 개수 | Number | 페이지 응답일 때 | X | 25 |
| page | 현재 페이지 | Number | 페이지 응답일 때 | X | 1 |
| page_size | 페이지 크기 | Number | 페이지 응답일 때 | X | 10 |

| results | 진단 결과 목록 | Array |  | X | [] |

| status | response content |
