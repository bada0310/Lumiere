# JWT 로그인

Method: POST
URL: /accounts/jwt-login/
param: password, username
사용자: 비회원
설명: JWT access, refresh 토큰을 발급합니다.
순서: 4
인증필요여부: 인증 불필요
카테고리: 계정 11

### Request Syntax

```jsx
POST /accounts/jwt-login/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `username` | 확인 필요 | 확인 필요 | username 값 |
| `password` | 확인 필요 | 확인 필요 | password 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| username | 아이디 | String |  | X | "user01" |
| password | 비밀번호 | String |  | X | "password1234" |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| refresh | refresh token | String |  | X | "refresh_token" |
| access | access token | String |  | X | "access_token" |

### Status

| status | response content |
| --- | --- |
| 200 | JWT 토큰 발급 성공 |
| 400 | 요청 값 누락 |
| 401 | 로그인 실패 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "refresh": "refresh_token",
  "access": "access_token"
}
```
