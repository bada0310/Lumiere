# 메인 퍼스널컬러 설정 취소

Method: POST
URL: /api/diagnosis/results/{result_id}/unset-primary/
param: result_id
사용자: 회원
설명: 선택한 진단 결과의 메인 설정을 취소합니다.
순서: 20
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
POST /api/diagnosis/results/{result_id}/unset-primary/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `result_id` | 확인 필요 | 확인 필요 | result_id 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |

### Request Body

요청 바디 없음.

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 진단 결과 ID | Number |  | X | 1 |
| is_primary | 메인 진단 여부 | Boolean |  | X | false |
| message | 처리 결과 메시지 | String |  | X | "메인 퍼스널컬러 설정이 취소되었습니다." |

### Status

| status | response content |
| --- | --- |
| 200 | 설정 취소 성공 |
| 401 | 인증 실패 |
| 404 | 진단 결과 없음 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "id": 1,
  "is_primary": false,
  "message": "메인 퍼스널컬러 설정이 취소되었습니다."
}
```
