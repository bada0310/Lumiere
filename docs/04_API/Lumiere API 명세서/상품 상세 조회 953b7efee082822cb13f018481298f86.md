# 상품 상세 조회

Method: GET
URL: /api/products/{product_id}/
param: product_id, second_tone_key, tone_key
사용자: 전체
설명: 특정 상품 상세와 옵션별 추천 정보를 조회합니다.
순서: 25
인증필요여부: 인증 선택
카테고리: 상품/추천 21

### Request Syntax

```jsx
GET /api/products/{product_id}/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `product_id` | 확인 필요 | 확인 필요 | product_id 값 |
| `tone_key` | 확인 필요 | 확인 필요 | tone_key 값 |
| `second_tone_key` | 확인 필요 | 확인 필요 | second_tone_key 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | X | 로그인 사용자는 개인화 추천에 활용 가능 | Bearer {access_token} |

### Query parameter

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| tone_key | tone_key 값 | 확인 필요 |  | O | 확인 필요 |
| second_tone_key | second_tone_key 값 | 확인 필요 |  | O | 확인 필요 |

### Response

상품 객체.

### Status

| status | response content |
| --- | --- |
| 200 | 상세 조회 성공 |
| 400 | 잘못된 query parameter |
| 404 | 상품 없음 |
| 500 | 서버 내부 오류 |
