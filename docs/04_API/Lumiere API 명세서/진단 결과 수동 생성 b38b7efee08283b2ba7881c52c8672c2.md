# 진단 결과 수동 생성

Method: POST
URL: /api/diagnosis/results/
param: confidence_score, personal_color_id, status, summary, tone_key
사용자: 회원
설명: Serializer 기반으로 진단 결과를 직접 생성합니다.
순서: 16
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
POST /api/diagnosis/results/
```

### Parameters

| Name | Type | Required | Description |

| `tone_key` | 확인 필요 | 확인 필요 | tone_key 값 |

| `summary` | 확인 필요 | 확인 필요 | summary 값 |

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
