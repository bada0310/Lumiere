# 상품 목록 조회

Method: GET
URL: /api/products/
param: category, q, second_tone_key, tone_key
사용자: 전체
설명: 상품 목록을 조회하고 검색/카테고리/톤 기준으로 필터링합니다.
순서: 24
인증필요여부: 인증 선택
카테고리: 상품/추천 21

### Request Syntax

```jsx
GET /api/products/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `category` | 확인 필요 | 확인 필요 | category 값 |
| `q` | 확인 필요 | 확인 필요 | q 값 |
| `tone_key` | 확인 필요 | 확인 필요 | tone_key 값 |
| `second_tone_key` | 확인 필요 | 확인 필요 | second_tone_key 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | X | 로그인 사용자는 개인화 추천에 활용 가능 | Bearer {access_token} |

### Query parameter

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| category | category 값 | 확인 필요 |  | O | 확인 필요 |
| q | q 값 | 확인 필요 |  | O | 확인 필요 |
| tone_key | tone_key 값 | 확인 필요 |  | O | 확인 필요 |
| second_tone_key | second_tone_key 값 | 확인 필요 |  | O | 확인 필요 |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 상품 ID | Number |  | X | 1 |
| brand | 브랜드명 | String |  | O | "브랜드명" |
| name | 상품명 | String |  | X | "상품명" |
| category | 카테고리 | String(ENUM) |  | X | "LIP" |
| image_url | 상품 이미지 URL | String |  | O | "https://..." |
| product_url | 상품 상세 URL | String |  | O | "https://..." |
| price | 가격 | Number |  | O | 15000 |
| match_score | 추천 점수 | Number |  | O | 87 |
| reason | 추천 사유 | String |  | O | "추천 사유" |
| options | 상품 옵션 목록 | Array |  | O | [] |

### Status

| status | response content |
| --- | --- |
| 200 | 상품 목록 조회 성공 |
| 400 | 잘못된 query parameter |
| 500 | 서버 내부 오류 |
