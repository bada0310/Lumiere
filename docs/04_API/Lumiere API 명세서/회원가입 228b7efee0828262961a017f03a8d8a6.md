# 회원가입

Method: POST
URL: /accounts/signup/
param: email, password, username
사용자: 비회원
설명: 신규 사용자를 생성합니다.
순서: 1
인증필요여부: 인증 불필요
카테고리: 계정 11

### Request Syntax

```jsx
POST /accounts/signup/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `username` | 확인 필요 | 확인 필요 | username 값 |
| `password` | 확인 필요 | 확인 필요 | password 값 |
| `email` | 확인 필요 | 확인 필요 | email 값 |

### Request Header

| **Header** | **필수** | 설명 | **예시** |
| --- | --- | --- | --- |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| username | 아이디 | String |  | X | "user01" |
| password | 비밀번호 | String |  | X | "password1234" |
| email | 이메일 | String |  | O | "user01@example.com" |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| message | 처리 결과 메시지 | String |  | X | "회원가입이 완료되었습니다.” |

### Status

| status | response content |
| --- | --- |
| 201 | 회원가입 성공 |
| 400 | 필수 값 누락, 중복 아이디, 유효하지 않은 입력 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "message": "회원가입이 완료되었습니다."
}
```
