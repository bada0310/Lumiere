# 상품 이미지 색상 분석

Method: POST
URL: /api/products/analyze-image/
param: brand_name, category, image, product_name
사용자: 회원
설명: 사용자가 업로드한 상품 이미지에서 색상 후보와 추천 점수를 분석합니다.
순서: 29
인증필요여부: 로그인 필요
카테고리: 상품/추천 21

### Request Syntax

```jsx
POST /api/products/analyze-image/
```

| `brand_name` | 확인 필요 | 확인 필요 | brand_name 값 |
| `category` | 확인 필요 | 확인 필요 | category 값 |
