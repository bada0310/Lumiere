# AI 퍼스널컬러 진단 실행

Method: POST
URL: /api/diagnosis/analyze/
param: image
사용자: 회원
설명: 얼굴 이미지를 업로드해 AI 퍼스널컬러 진단 결과를 생성합니다.
순서: 12
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
POST /api/diagnosis/analyze/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `image` | 확인 필요 | 확인 필요 | image 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |
| Content-Type | O | 파일 업로드 요청 | multipart/form-data |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| image | image 값 | 확인 필요 |  | 확인 필요 | 확인 필요 |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 진단 결과 ID | Number |  | X | 1 |
| status | 진단 상태 | String(ENUM) | completed, low_confidence, failed | X | "completed" |
| tone_key | 퍼스널컬러 키 | String |  | X | "summer_cool_mute" |
| tone_label | 화면 표시용 톤 이름 | String |  | O | "여름 쿨 뮤트" |
| confidence | 신뢰도 | Number | 0~100 | O | 82 |
| diagnosed_at | 진단 일자 | String(Date) |  | O | "2026-06-24" |
| summary | 진단 요약 | String |  | O | "진단 요약" |
| uploaded_image_url | 업로드 이미지 URL | String |  | O | "http://127.0.0.1:8000/media/..." |
| skin_metrics | 피부 분석 지표 | Object |  | O | {} |
| representative_colors | 대표 색상 | Array |  | O | [] |
| color_palettes | 추천 팔레트 | Object |  | O | {} |
| makeover_styles | 메이크오버 스타일 | Array |  | O | [] |
| makeup_generation_status | 메이크오버 생성 상태 | String(ENUM) | none, queued, running, complete, failed, skipped | X | "none" |
| is_primary | 메인 진단 여부 | Boolean |  | X | true |

### Status

| status | response content |
| --- | --- |
| 201 | 진단 결과 생성 성공 |
| 400 | 이미지 누락, 진단 결과 검증 실패 |
| 401 | 인증 실패 |
| 413 | 파일 크기 초과 |
| 502 | AI 응답 파싱 실패 |
| 503 | AI 설정 누락 |
