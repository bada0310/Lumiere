# 상품 Router 생성

Method: POST
URL: /api/products/items/
param: brand, category, image_url, name, product_url 등
사용자: 회원
설명: 상품 데이터를 생성합니다.
순서: 36
인증필요여부: 로그인 필요
카테고리: 상품/추천 21

### Request Syntax

```jsx
POST /api/products/items/
```

### Parameters

| Name | Type | Required | Description |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| `brand` | 확인 필요 | 확인 필요 | brand 값 |  |  |  |
| `name` | 확인 필요 | 확인 필요 | name 값 |  |  |  |
| `category` | 확인 필요 | 확인 필요 | category 값 |  |  |  |
| `image_url` | 확인 필요 | 확인 필요 | image_url 값 |  |  |  |
| `product_url` 등` | 확인 필요 | 확인 필요 | product_url` 등 값 |  |  |  |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| brand | 브랜드명 | String |  | O | "브랜드명" |
| name | 상품명 | String |  | X | "상품명" |
| category | 카테고리 | String(ENUM) |  | X | "LIP" |
| image_url | 상품 이미지 URL | String |  | O | "https://..." |
| product_url | 상품 상세 URL | String |  | O | "https://..." |
| price | 가격 | Number |  | O | 15000 |

### Response

생성된 상품 객체.

### Status

| status | response content |
| --- | --- |
| 201 | 상품 생성 성공 |
| 400 | 필수 값 누락 또는 유효하지 않은 값 |
| 401 | 인증 실패 |
| 500 | 서버 내부 오류 |
