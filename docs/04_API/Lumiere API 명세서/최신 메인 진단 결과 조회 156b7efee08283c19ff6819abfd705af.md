# 최신 메인 진단 결과 조회

Method: GET
URL: /api/diagnosis/latest/
param: -
사용자: 회원
설명: 로그인 사용자의 메인 퍼스널컬러 진단 결과를 조회합니다.
순서: 14
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
GET /api/diagnosis/latest/
```

### Parameters

없음.

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |

### Request Body

요청 바디 없음.

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 진단 결과 ID | Number |  | X | 1 |
| tone_key | 퍼스널컬러 키 | String |  | X | "summer_cool_mute" |
| is_primary | 메인 진단 여부 | Boolean |  | X | true |

### Status

| status | response content |
| --- | --- |
| 200 | 조회 성공 |
| 401 | 인증 실패 |
| 500 | 서버 내부 오류 |

### Example

```jsx
{
  "id": 1,
  "tone_key": "summer_cool_mute",
  "is_primary": true
}
```
