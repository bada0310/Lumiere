# 회원 탈퇴

Method: DELETE
URL: /accounts/user/, /accounts/user/delete/
param: confirmation, password
사용자: 회원
설명: 확인 문구와 비밀번호 검증 후 회원을 탈퇴 처리합니다.
순서: 8
인증필요여부: 로그인 필요
카테고리: 계정 11

### Request Syntax

```jsx
DELETE /accounts/user/, /accounts/user/delete/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `confirmation` | 확인 필요 | 확인 필요 | confirmation 값 |
| `password` | 확인 필요 | 확인 필요 | password 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |
| Content-Type | O | JSON 요청 | application/json |

### Request Body

요청 바디 없음.

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| message | 처리 결과 메시지 | String |  | X | "회원탈퇴가 완료되었습니다." |

### Status

| status | response content |
| --- | --- |
| 200 | 회원 탈퇴 성공 |
| 400 | 확인 문구 오류 또는 비밀번호 누락 |
| 401 | 인증 실패 |
| 403 | 비밀번호 불일치 |
| 500 | 서버 내부 오류 |
