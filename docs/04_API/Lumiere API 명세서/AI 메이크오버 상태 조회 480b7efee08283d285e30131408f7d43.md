# AI 메이크오버 상태 조회

Method: GET
URL: /api/diagnosis/results/{result_id}/makeovers/
param: result_id
사용자: 회원
설명: 진단 결과의 메이크오버 생성 상태와 스타일별 이미지를 조회합니다.
순서: 21
인증필요여부: 로그인 필요
카테고리: 퍼스널컬러 진단 12

### Request Syntax

```jsx
GET /api/diagnosis/results/{result_id}/makeovers/
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `result_id` | 확인 필요 | 확인 필요 | result_id 값 |

### Request Header

| Header | 필수 | 설명 | 예시 |
| --- | --- | --- | --- |
| Authorization | O | JWT access token | Bearer {access_token} |

### Request Body

요청 바디 없음.

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 전체 메이크오버 상태 | String(ENUM) | queued, running, complete, failed | X | "queued" |
| styles | 스타일별 메이크오버 목록 | Array |  | X | [] |
| error | 에러 메시지 | String |  | O | "" |

### Status

| status | response content |
| --- | --- |
| 200 | 상태 조회 성공 |
| 401 | 인증 실패 |
| 404 | 진단 결과 없음 |
| 500 | 서버 내부 오류 |
